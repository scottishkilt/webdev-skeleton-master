webdev-skeleton
===============

Getting started:

0. Get and install [npm](https://www.npmjs.org/doc/README.html), the [node.js](http://nodejs.org/download/) package manager.
1. Use npm to install Grunt: (sudo) `npm install -g grunt-cli`
2. Clone this repository if you haven't already
2. Change directory to the root of this project, and install dependencies with `npm install`
3. Still at the project root, run a webserver with `grunt server`
4. Open a text editor and edit `index.html`, `js/main.js`, and (optionally) `css/main.css`
